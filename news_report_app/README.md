# تطبيق تقارير الأخبار - News Report Application

## نظرة عامة
تطبيق مكتبي بايثون يتيح جمع عدة عناصر أخبار/تقارير بشكل تفاعلي، وتنسيقها بشكل احترافي، وتصديرها إلى ملف Microsoft Word (.docx) مع دعم كامل للغة العربية من اليمين إلى اليسار (RTL).

## البنية التحتية
- **App**: واجهة المستخدم الرسومية باستخدام `customtkinter`
- **NewsManager**: إدارة بيانات الأخبار في الذاكرة
- **DocExporter**: منطق إنشاء مستندات Word مع تنسيق عربي صارم

## متطلبات التشغيل
- Python 3.10 أو أعلى
- Windows 10/11 (مُخصص للبيئة العربية)

## التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
cd news_report_app
pip install -r requirements.txt
```

### 2. تشغيل التطبيق
```bash
python main.py
```

## كيفية الاستخدام
1. اضغط على "➕ إضافة خبر جديد" لفتح نافذة إدخال الخبر
2. املأ الحقول المطلوبة: المصدر، العنوان، والمحتوى
3. أضف المرفقات الاختيارية: صورة، إحداثيات، أو توصية
4. اضغط "✅ حفظ الخبر" لإضافته إلى القائمة
5. عاين القائمة في جدول الأخبار
6. اضغط "🗑️ حذف المحدد" لحذف أي خبر
7. اضغط "📄 إنشاء التقرير" لتصدير ملف Word

## ملاحظات التنسيق العربي
- جميع الفقرات في مستند Word مُنسقة كـ RTL (يمين إلى يسار)
- تستخدم خطوط عربية: Traditional Arabic, Simplified Arabic, Arial
- تم تطبيق خصائص RTL عبر XML مباشرة لضمان التوافق مع Word
- تم اختبار عرض الحروف العربية والعلامات والتماثيل

## استكشاف الأخطاء
- **الخطوط غير مرئية**: تأكد من تثبيت خطوط عربية على النظام
- **تحذير توافق التخطيط**: تأكد من فتح الملف مباشرةا من خلال Word (ليس WordPad)
- **الصور غير المرئية**: تحقق من صيغة الصورة (PNG/JPG) ومسارها

## الملفات المُنشأة
- `output/تقرير_الأخبار_YYYYMMDD_HHMM.docx` - ملف التقرير المُنشأ
