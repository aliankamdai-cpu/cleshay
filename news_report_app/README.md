# تطبيق تقارير الأخبار - News Report Application

## نظرة عامة
تطبيق مكتبي بايثون يتيح جمع عدة عناصر أخبار/تقارير بشكل تفاعلي، وتنسيقها بشكل احترافي، وتصديرها إلى ملف Microsoft Word (.docx) مع دعم كامل للغة العربية من اليمين إلى اليسار (RTL).

## البنية التحتية
- **App**: واجهة المستخدم الرسومية باستخدام `customtkinter`
- **NewsManager**: إدارة بيانات الأخبار في الذاكرة
- **DocExporter**: منطق إنشاء مستندات Word مع تنسيق عربي صارم

## متطلبات التشغيل
- Python 3.10 أو أعلى
- Windows 10/11 (مُخصص للبيئة العربية)

## التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
cd news_report_app
pip install -r requirements.txt
```

### 2. تشغيل التطبيق
```bash
python main.py
```

## كيفية الاستخدام
1. اضغط على "➕ إضافة خبر جديد" لفتح نافذة إدخال الخبر
2. املأ الحقول المطلوبة: المصدر، العنوان، والمحتوى
3. اختر تصنيف الخبر من القائمة المنسدلة (سياسة، اقتصاد، رياضة، إلخ)
4. أضف المرفقات الاختيارية: صورة، إحداثيات، أو توصية
5. اضغط "✅ حفظ الخبر" لإضافته إلى القائمة
6. عاين القائمة في جدول الأخبار
7. **اختر الأخبار للتقرير**: انقر على عمود "✓" لتحديد/إلغاء تحديد كل خبر
8. **بدائل التحديد**: 
   - انقر مرتين على أي خبر لتبديل حالته
   - انقر بالزر الأيمن للوصول إلى خيارات التحديد (تحديد الكل، إلغاء التحديد)
9. اضغط "🗑️ حذف المحدد" لحذف أي خبر
10. اضغط "📄 توليد التقرير Word" لتصدير ملف Word بالأخبار المحددة فقط

## ميزات متقدمة
- **التصنيف**: كل خبر يُصنف حسب نوعه (عام، سياسة، اقتصاد، رياضة، تكنولوجيا، صحة، تعليم، ثقافة، منوعات، حوادث، طقس)
- **التحديد الانتقائي**: اختر الأخبار التي تريد تضمينها في التقرير النهائي فقط
- **التجميع التلقائي**: يتم ترتيب الأخبار في التقرير حسب التصنيف، مع عرض جميع أخبار كل تصنيف معًا
- **تنسيق احترافي**: خط Calibri، مسافة أسطر 1.5، صور بمحاذاة النص، عناوين كبيرة واضحة

## ملاحظات التنسيق العربي
- جميع الفقرات في مستند Word مُنسقة كـ RTL (يمين إلى يسار)
- تستخدم خطوط عربية: Calibri (الخط الأساسي)
- تم تطبيق خصائص RTL عبر XML مباشرة لضمان التوافق مع Word
- تم اختبار عرض الحروف العربية والعلامات والتماثيل
- المسافة بين الأسطر 1.5 لجميع النصوص
- الصور تظهر بمحاذاة النص (يمين للنصوص العربية)

## استكشاف الأخطاء
- **الخطوط غير مرئية**: تأكد من تثبيت خطوط عربية على النظام
- **تحذير توافق التخطيط**: تأكد من فتح الملف مباشرةا من خلال Word (ليس WordPad)
- **الصور غير المرئية**: تحقق من صيغة الصورة (PNG/JPG) ومسارها
- **Ctrl+V لا يعمل**: تم تثبيت pyperclip لدعم اللصق بشكل أفضل
